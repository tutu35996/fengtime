# 风云时间转换器

一个功能强大的多时区时间转换工具，支持固定时区和实时查询，专为全球团队协作设计。

## ✨ 主要功能

- 🌍 **多时区支持**：支持全球200+个国家和地区
- ⏰ **实时查询**：自动获取夏令时变化
- 🔒 **固定时区**：无夏令时国家的稳定时间显示
- 🎯 **预设模板**：快速配置常用时区组合
- 🔄 **一键刷新**：实时更新所有时间
- 📱 **响应式设计**：完美适配各种设备

## 🚀 快速开始

### 在线使用
直接访问 [GitHub Pages](https://YOUR_USERNAME.github.io/fengtime) 即可使用

### 本地运行
1. 克隆仓库
```bash
git clone https://github.com/your-username/fengtime.git
cd fengtime
```

2. 打开 `index.html` 文件即可使用

## 🎯 使用指南

### 基本操作
1. **选择主国家**：设置基准时区
2. **输入时间**：输入需要转换的时间
3. **添加目标国家**：选择要转换到的时区
4. **查看结果**：自动显示转换后的时间

### 预设模板
- **ZUES 欧洲**：欧洲主要时区
- **ZUES 美洲**：美洲主要时区
- **AG**：迪拜时间模板
- **DCG**：美国东部时间模板
- **Two Sigma**：双子星时间模板
- **ORBIS 欧洲**：ORBIS欧洲时间模板
- **ORBIS 亚洲**：ORBIS亚洲时间模板

## 🔧 技术特性

### 时区处理
- **固定时区**：使用预定义UTC偏移，显示"无夏令时"
- **实时查询**：通过WorldTimeAPI获取最新时区信息，显示"实时查询"
- **智能缓存**：5分钟缓存机制，减少API调用
- **重试机制**：网络失败时自动重试3次

### 用户体验
- **视觉反馈**：刷新时显示"--:--"过渡动画
- **状态标识**：清楚显示数据来源类型
- **错误处理**：友好的错误提示和降级方案
- **响应式布局**：适配桌面和移动设备
